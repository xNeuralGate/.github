# НейроГейт.РФ

НейроГейт — российская ИИ-платформа для B2B.

Мы создаём инфраструктуру для корпоративной работы с ИИ-моделями: единый API, маршрутизацию между провайдерами, аудит, биллинг, лимиты, управление доступом и поддержку суверенного контура.

## Что входит в платформу

- **НейроГейт API** — OpenAI-совместимый ИИ-шлюз для доступа к нескольким моделям через единый интерфейс
- **НейроГейт Панель управления** — ключи, usage, биллинг, администрирование и контроль доступа
- **НейроГейт Чат** — корпоративная чат-платформа для команд и рабочих сценариев
- **НейроГейт Пилот** — ИИ-агент для разработчиков и продуктовых команд

## Для кого

НейроГейт предназначен для компаний, которым нужен предсказуемый, безопасный и управляемый способ встраивать ИИ в продукты, внутренние системы и клиентские сервисы.

## Ключевые возможности

- OpenAI-совместимый API
- маршрутизация между моделями и провайдерами
- прозрачный биллинг и лимиты
- аудит и журналирование
- суверенный контур для корпоративных сценариев
- единый слой интеграции для B2B-внедрений
